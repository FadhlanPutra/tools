# build.my.id

Tools online gratis — React + TypeScript + Vite + Tailwind CSS

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy ke Cloudflare Pages

1. Push ke GitHub
2. Connect repo di [Cloudflare Pages](https://pages.cloudflare.com)
3. Build command: `npm run build`
4. Output directory: `dist`
